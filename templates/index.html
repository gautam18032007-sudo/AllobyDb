Table schema — products

ColumnTypeNotes
idSERIALAuto-increment primary key
nameTEXTProduct name (NOT NULL)
categoryTEXTElectronics · Kitchen · Sports · Furniture · Home
priceNUMERIC(10,2)USD price, range $19.99–$329.00
stockINTEGERUnits in stock, range 12–150
ratingNUMERIC(3,1)Customer rating 0.0–5.0
descriptionTEXTShort product description
created_atTIMESTAMPTZRow creation timestamp (DEFAULT NOW())

Indexes

idx_products_category ON products(category)
idx_products_price ON products(price)
idx_products_rating ON products(rating)
pgvector extension ENABLED (AlloyDB AI-ready)

AI Pipeline

1. User types natural language question
2. Claude generates a safe SELECT query
3. SQL validated (SELECT-only guard)
4. Executed on AlloyDB via psycopg2
5. Claude summarises results in plain English
